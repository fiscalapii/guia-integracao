# Contribuindo

Obrigado por considerar contribuir com a documentacao da FiscalAPI!

## Como Contribuir

### Reportar Erros na Documentacao

Abra uma [issue](https://github.com/fiscalapii/guia-integracao/issues) descrevendo:
- Qual pagina tem o erro
- O que esta errado
- O que deveria ser

### Adicionar Exemplos

Quer adicionar um exemplo em uma linguagem que ainda nao temos? Otimo!

1. Fork este repositorio
2. Crie uma pasta em `exemplos/{linguagem}/`
3. Siga o padrao dos exemplos existentes:
   - Comentario no topo com link para fiscalapi.com.br
   - Use variaveis de ambiente para a API key
   - Inclua exemplos funcionais com `main`
4. Abra um Pull Request

### Melhorar a Documentacao

- Correcoes de portugues e clareza
- Adicionar exemplos de resposta
- Melhorar descricoes de campos

## Estilo

- Documentacao em portugues do Brasil
- Sem acentos nos nomes de arquivo
- Exemplos de codigo devem ser auto-contidos e executaveis
- Use dados ficticios (nunca CPF/CNPJ reais)

## Duvidas

Abra uma issue ou envie um email para contato@fiscalapi.com.br.
